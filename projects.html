Twitch Bot

Twitch Moderation Bot

This was my Twitch Moderation Bot which i coded in JavaScript I coded this as a demo for a Twitch Streamer to see what they would think to a bot moderator over a human moderator. How I coded this was, I first thing I did was to create a application on the Twitch developer portal. I then created my project in vscode, the next thing I did was open a windows powershell, navigate to the folder to where the project is then type npm innit. This creates a .json file with the bots name, description, version, etc. I then downloaded yarn into my project using npm yarn. I used this to add esm then loaded this in my index file. I then added another file where I would do all the bots features, first of all I added console.log('test'). This command outputed the work test if everything worked properly which it did. I added tmi.js next. What this is, is a package that allows the bot to communicate with Twitch, I went on the documentation and copied and pasted the code to get it started as this ensures that it is setup correct. I then added tmi.js to my .json file with yarn (yarn add tmi.js). I then decided to get the OAuth token which allows my to connect to twitch. Using a url and changing parameters I got my OAuth token. I then inputted this into my code. I again typed yarn start to see if this worked which it did. I then went onto programming in my features.
